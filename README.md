# Expense Tracker App

## Overview
This is a simple expense tracker web application built using Next.js. It helps users manage their finances by tracking income and expenses, and displaying the current balance in a clean interface.

## Features
- Add income and expenses
- Categorize transactions
- View transaction history
- Track total balance in real time
- Simple and user-friendly interface

## Tech Stack
- Next.js
- React
- JavaScript
